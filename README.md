
This project collects discount deals from convenience stores. In the absence of an official API, I scraped the data directly using a crawler and integrated the CSV-to-SQL workflow into a Flask backend. The frontend features an interactive map powered by the Kakao Maps API, where I mixed JS and HTML to enable navigation services and a location-based search for nearby convenience stores via button clicks.
